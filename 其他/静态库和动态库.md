| 特性       | 静态库      | 动态库        |
| -------- | -------- | ---------- |
| **链接时机** | 编译时链接    | 运行时加载      |
| **文件大小** | 可执行文件较大  | 可执行文件较小    |
| **内存占用** | 每个程序独立拷贝 | 多个程序共享一份   |
| **更新维护** | 需要重新编译   | 只需替换库文件    |
| **依赖问题** | 无依赖问题    | 需要库文件存在    |
| **启动速度** | 启动快      | 启动稍慢（需加载库） |

`windows`静态库和动态库的命名特点：
1. **静态库**：只有 `.lib`，包含代码
2. **动态库**：有 `.dll` + `.lib`（导入库）
