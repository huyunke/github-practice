- `MSG` 结构体是 Windows 系统内置的。它定义在 Windows SDK 的头文件（通常是 `WinUser.h`，由 `Windows.h` 自动包含）中
- 这是所有消息传递的基础。每一个 `MSG` 变量都是一个装满信息的结构体

|**字段**|**含义**|
|---|---|
|**`hwnd`**|接收消息的窗口句柄（送往哪家）。|
|**`message`**|消息标识符（如 `WM_PAINT`, `WM_LBUTTONDOWN`）。|
|**`wParam`**|附加信息 1（通常是按键码或状态）。|
|**`lParam`**|附加信息 2（通常包含鼠标坐标等复杂数据）。|
|**`time`**|消息投递的时间戳。|
|**`pt`**|消息发生时，鼠标在屏幕上的物理坐标。|
