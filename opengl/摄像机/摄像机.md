## 摄像机包含的属性
### 摄像机位置
- 简单来说就是世界空间中一个指向摄像机位置的向量
- 正z轴是从屏幕指向你的，如果我们希望摄像机向后移动，我们就沿着z轴的正方向移动
- 定义一个向量来表示摄像机的位置
```cpp
//值随意
glm::vec3 cameraPos = glm::vec3(0.0f, 0.0f, 3.0f);
```

### 摄像机方向
- 定义一个向量来表示摄像机的位置
```cpp
// cameraTarget - 摄像机目标点，定义了摄像机注视的位置
glm::vec3 cameraTarget = glm::vec3(0.0f, 0.0f, 0.0f); 
// 从目标点指向摄像机的向量
glm::vec3 cameraDirection = glm::normalize(cameraPos - cameraTarget);
```

### 右轴
- 表示摄像机空间的x轴的正方向
```cpp
glm::vec3 up = glm::vec3(0.0f, 1.0f, 0.0f); 
glm::vec3 cameraRight = glm::normalize(glm::cross(up, cameraDirection));
```

### 上轴
右方向和方向向量进行叉乘
```cpp
glm::vec3 cameraUp = glm::cross(cameraDirection, cameraRight);
```

## Look At
- LookAt矩阵会创建一个看着(Look at)给定目标的观察矩阵
```cpp
glm::mat4 view; 
view = glm::lookAt(glm::vec3(0.0f, 0.0f, 3.0f), 
					glm::vec3(0.0f, 0.0f, 0.0f), 
					glm::vec3(0.0f, 1.0f, 0.0f));
```
- 第一个参数：摄像机的位置
- 第二个参数：摄像机指向的目标
- 第三个参数：上向量

## 移动速度
- 图形程序和游戏通常会跟踪一个时间差变量，它储存了渲染上一帧所用的时间。我们把所有速度都去乘以`deltaTime`值
- 通过跟踪两个全局变量来计算出`deltaTime`值
```cpp
float deltaTime = 0.0f; // 当前帧与上一帧的时间差 
float lastFrame = 0.0f; // 上一帧的时间
```
- 每一帧我们都计算出新的`deltaTime`以备后用
```cpp
float currentFrame = glfwGetTime(); 
deltaTime = currentFrame - lastFrame; 
lastFrame = currentFrame;
```
在计算摄像机速度时将其考虑进去
```cpp
void processInput(GLFWwindow *window) { 
	float cameraSpeed = 2.5f * deltaTime; 
	... 
}
```

## 欧拉角
- **俯仰角**是描述我们如何往上或往下看的角
- **偏航角**表示我们往左和往右看的程度
- **滚转角**代表我们如何**翻滚**摄像机
