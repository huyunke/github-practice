## 图形学中如何计算出反射颜色
我们将这两个颜色向量作分量相乘，结果就是最终的颜色向量了

## 风氏光照模型
- 环境光照(Ambient Lighting)：即使在黑暗的情况下，世界上通常也仍然有一些光亮（月亮、远处的光），所以物体几乎永远不会是完全黑暗的。使用一个环境光照常量模拟
- 漫反射光照(Diffuse Lighting)：模拟光源对物体的方向性影响(Directional Impact)。它是风氏光照模型中视觉上最显著的分量。物体的某一部分越是正对着光源，它就会越亮
- 镜面光照(Specular Lighting)：模拟有光泽物体上面出现的亮点。镜面光照的颜色相比于物体的颜色会更倾向于光的颜色


## 环境光照(ambient)
通过用光的颜色(`lightColor`)乘以一个很小的常量环境因子(`ambientStrength`)，再乘以物体的颜色(`objectColor`)，然后将最终结果作为片段的颜色(`FragColor`)
- 片段着色器
```
void main() { 
	float ambientStrength = 0.1; 
	vec3 ambient = ambientStrength * lightColor; 
	vec3 result = ambient * objectColor; 
	FragColor = vec4(result, 1.0); 
}
```
## 漫反射光照(Diffuse)
### 如何计算漫反射光照
以下向量在计算时全部都标准化为单位向量
- 法向量（`Normal`）：一个垂直于顶点表面的向量
- 定向的光线（`lightDir`）：作为光源的位置与片段的位置之间向量差的方向向量。为了计算这个光线，我们需要光的位置向量和片段的位置向量

### 代码
- 顶点着色器
```
#version 460 core
layout (location=0) in vec3 aPos;
layout (location=1) in vec3 aNormal;//法线数据

out vec3 Normal;
out vec3 FragPos;

uniform mat4 model; 
uniform mat4 view; 
uniform mat4 projection;

void main(){
	// 将顶点位置转换到世界空间，用于计算光照方向 
	FragPos = vec3(model * vec4(aPos, 1.0)); 
	// 核心：使用法线矩阵转换法线，防止非等比缩放破坏法线方向 
	// 如果没有非等比缩放，可以直接用 mat3(model) * aNormal 
	// 由于矩阵求逆是一项对于着色器开销很大的运算，所以应尽量避免在着色器器中进行求逆运算，最好现在CPU上计算出发现矩阵，再通过uniform把它传递给着色器
	// Normal = mat3(transpose(inverse(model))) * aNormal; 
	gl_Position = projection * view * vec4(FragPos, 1.0);
}
```
- 片段着色器（结合了环境光照和漫反射光照）
```
#version 330 core
out vec4 FragColor;

in vec3 Normal;   // 来自顶点着色器
in vec3 FragPos;  // 来自顶点着色器

uniform vec3 lightPos;   // 光源的世界坐标
uniform vec3 lightColor; // 光源颜色
uniform vec3 objectColor;// 物体颜色

void main()
{
    // 1. 环境光 (Ambient) - 通常会给一点基础亮度防止全黑
    float ambientStrength = 0.1;
    vec3 ambient = ambientStrength * lightColor;
  	
    // 2. 漫反射 (Diffuse)
    vec3 norm = normalize(Normal);          // 标准化法线
    vec3 lightDir = normalize(lightPos - FragPos); // 计算片段指向光源的方向
    
    // 计算点积，使用 max 确保夹角大于 90 度时亮度不为负数
    float diff = max(dot(norm, lightDir), 0.0);
    vec3 diffuse = diff * lightColor;
            
    // 3. 合并结果
    vec3 result = (ambient + diffuse) * objectColor;
    FragColor = vec4(result, 1.0);
}
```

## 镜面光照（Specular）
### 如何计算
镜面光照由**光的方向向量**，**物体的法向量**和**观察方向**共同决定（使用观察者的世界空间位置和片段的位置来计算观察向量）

### 代码
1. 更新`uniform`，把相机位置（即观察者位置）传给片段着色器，把反光度传给片段着色器
```cpp
lightingShader.setVec3("viewPos", camera.Position);
lightingShader.setFloat("shininess", 32.0f);
```
2. 片段着色器
```OpenGL Shading Language
#version 460 core

uniform vec3 viewPos;     // 相机位置
uniform float shininess;  // 反光度（通常取 32, 64, 128 等）

void main() {
    // 1. 环境光 (Ambient) ...
    
    // 2. 漫反射 (Diffuse) ...

    // 3. 镜面光 (Specular)
    float specularStrength = 0.5; // 镜面强度
    vec3 viewDir = normalize(viewPos - FragPos); // 观察方向
    vec3 reflectDir = reflect(-lightDir, norm);  // 反射方向（注意 lightDir 要取反）
    
    // 计算镜面分量：观察方向与反射方向越接近，高光越强
    float spec = pow(max(dot(viewDir, reflectDir), 0.0), shininess);
    vec3 specular = specularStrength * spec * lightColor;  

    // 最终颜色合成
    vec3 result = (ambient + diffuse + specular) * objectColor;
    FragColor = vec4(result, 1.0);
}
```
`shininess`：决定了高光的散射范围
- **数值越小（如 2 或 4）**：高光点很大、很散，看起来像粗糙的塑料或木头
- **数值越大（如 128 或 256）**：高光点非常小且锐利，看起来像抛光的金属或镜面
