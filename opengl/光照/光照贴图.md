## 漫反射贴图
将纹理储存为Material结构体中的一个`sampler2D`。将之前定义的`vec3`漫反射颜色向量替换为漫反射贴图
- 注意`sampler2D`是所谓的不透明类型，也就是说我们不能将它实例化，只能通过uniform来定义它。如果我们使用除uniform以外的方法（比如函数的参数）实例化这个结构体，GLSL会抛出一些奇怪的错误。这同样也适用于任何封装了不透明类型的结构体
- 环境光和漫反射共用一张贴图

### 片段着色器的修改
在光照计算中，漫反射贴图直接取代了原先结构体中的 `material.diffuse` 向量
### 计算逻辑的转变：

- **以前**：`结果 = 光的颜色 * (漫反射系数 * 固定的物体颜色)`
    
- **现在**：`结果 = 光的颜色 * (漫反射系数 * 从贴图中采样的颜色)`
    

> **采样（Sampling）**：这是一个关键动作。GPU 会根据顶点的 UV 坐标，去贴图里“查”出对应位置的颜色

## 镜面贴图
### 核心原理：黑白决定亮度

镜面贴图通常是一张灰度图（或者是彩色图中提取出的亮度信息）：

- **纯黑色 ($0.0$)**：代表该区域完全不反光（如缝隙中的泥土、粗糙的木头）
    
- **纯白色 ($1.0$)**：代表该区域有最强的镜面反射（如抛光的金属边框）
    
- **灰色中间值**：代表不同程度的反光
### 片段着色器的修改
在 `Material` 结构体中再添加一个采样器，并用它替换掉原先固定的 `material.specular` 向量

## 整合后的代码
### 顶点着色器
```OpenGL Shading Language
#version 460 core
layout (location = 0) in vec3 aPos;
layout (location = 1) in vec3 aNormal;
layout (location = 2) in vec2 aTexCoords;

out vec3 FragPos;
out vec3 Normal;
out vec2 TexCoords;

uniform mat4 model;
uniform mat4 view;
uniform mat4 projection;

void main() {
    // 将顶点位置转为世界空间
    FragPos = vec3(model * vec4(aPos, 1.0));
    // 使用法线矩阵变换法线，防止非等比缩放导致法线畸变
    Normal = mat3(transpose(inverse(model))) * aNormal;
    // 传递纹理坐标
    TexCoords = aTexCoords;
    
    gl_Position = projection * view * vec4(FragPos, 1.0);
}
```
### 片段着色器
```OpenGL Shading Language
#version 460 core
out vec4 FragColor;

struct Material {
    sampler2D diffuse;   // 漫反射贴图 (控制底色)
    sampler2D specular;  // 镜面贴图 (控制哪里亮，哪里暗)
    float shininess;     // 高光散射半径
}; 

struct Light {
    vec3 position;
    vec3 ambient;
    vec3 diffuse;
    vec3 specular;
};

in vec3 FragPos;  
in vec3 Normal;  
in vec2 TexCoords;

uniform vec3 viewPos;
uniform Material material;
uniform Light light;

void main() {
    // 【采样】从贴图中获取当前像素的数据
    vec3 diffMapColor = texture(material.diffuse, TexCoords).rgb;
    vec3 specMapColor = texture(material.specular, TexCoords).rgb;

    // 1. 环境光 (使用漫反射贴图颜色作为基准)
    vec3 ambient = light.ambient * diffMapColor;
  	
    // 2. 漫反射
    vec3 norm = normalize(Normal);
    vec3 lightDir = normalize(light.position - FragPos);
    float diff = max(dot(norm, lightDir), 0.0);
    vec3 diffuse = light.diffuse * (diff * diffMapColor);
    
    // 3. 镜面光 (关键变化：乘以采样到的镜面贴图颜色)
    vec3 viewDir = normalize(viewPos - FragPos);
    vec3 reflectDir = reflect(-lightDir, norm);  
    float spec = pow(max(dot(viewDir, reflectDir), 0.0), material.shininess);
    vec3 specular = light.specular * (spec * specMapColor);  
        
    // 最终颜色合成
    vec3 result = ambient + diffuse + specular;
    FragColor = vec4(result, 1.0);
}
```

### C++
```cpp
//激活并绑定漫反射贴图
//使用 GL_TEXTURE0
glActiveTexture(GL_TEXTURE0);
glBindTexture(GL_TEXTURE_2D, diffuseMap);
shader.setInt("material.diffuse", 0);

//激活并绑定镜面贴图
//使用 GL_TEXTURE1
glActiveTexture(GL_TEXTURE1);
glBindTexture(GL_TEXTURE_2D, specularMap);
shader.setInt("material.specular", 1);
```
