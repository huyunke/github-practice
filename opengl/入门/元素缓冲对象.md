## 创建
### 生成ID
```cpp
unsigned int EBO;
glGenBuffers(1, &EBO);
```
### 绑定目标
```cpp
glBindBuffer(GL_ELEMENT_ARRAY_BUFFER, EBO);
```
### 传输索引数据
```cpp
glBufferData(GL_ELEMENT_ARRAY_BUFFER, sizeof(indices), indices, GL_STATIC_DRAW);
```

**VAO（顶点数组对象）会记录 `GL_ELEMENT_ARRAY_BUFFER` 的绑定状态。**

- 这意味着如果你在绑定 VAO 的情况下绑定了 EBO，那么当你以后再次绑定这个 VAO 时，EBO 也会自动被绑定。这让你在渲染循环中只需要切换 VAO 就能完成所有配置
