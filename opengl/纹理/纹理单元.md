## 为什么需要
当你想要在同一个物体上**混合多张贴图**（比如木箱子贴上笑脸）时，你就需要同时把多张贴图“插进”不同的卡槽里

### 第一步：在着色器中定义“采样器” (Sampler) 

在片段着色器中，我们需要定义多个 `sampler2D` 变量

```
uniform sampler2D texture1; // 采样器1
uniform sampler2D texture2; // 采样器2

void main() {
    // 使用 mix 函数混合两个纹理，0.2 表示 80% 的来自 texture1，20% 来自 texture2
    FragColor = mix(texture(texture1, TexCoord), texture(texture2, TexCoord), 0.2);
}
```

### 第二步：将采样器绑定到纹理单元 (CPU 配置) 

告诉着色器，`texture1` 变量对应的是 `0` 号卡槽，`texture2` 对应的是 `1` 号卡槽

```cpp
ourShader.use(); // 必须先激活着色器程序，此处ourShader是封装好的着色器

// 告诉 OpenGL：名为 "texture1" 的采样器对应纹理单元 0
glUniform1i(glGetUniformLocation(ourShader.ID, "texture1"), 0); 
// 告诉 OpenGL：名为 "texture2" 的采样器对应纹理单元 1
glUniform1i(glGetUniformLocation(ourShader.ID, "texture2"), 1); 
```

### 第三步：渲染时激活并绑定纹理 

在渲染循环（`while` 循环）中，我们需要显式地把真正的纹理对象“塞进”对应的卡槽。

```cpp
glActiveTexture(GL_TEXTURE0);    // 1. 激活 0 号卡槽
glBindTexture(GL_TEXTURE_2D, texture1); // 2. 把 texture1 绑定到 0 号卡槽

glActiveTexture(GL_TEXTURE1);    // 1. 激活 1 号卡槽
glBindTexture(GL_TEXTURE_2D, texture2); // 2. 把 texture2 绑定到 1 号卡槽

// 然后进行正常的绘制
glBindVertexArray(VAO);
glDrawElements(GL_TRIANGLES, 6, GL_UNSIGNED_INT, 0);
```

1. **激活**：使用 `glActiveTexture` 选定卡槽
    
2. **绑定**：使用 `glBindTexture` 把贴图放进选定的卡槽
    
3. **指定**：使用 `glUniform1i` 告诉着色器里的 `sampler2D` 去哪个卡槽找贴图
