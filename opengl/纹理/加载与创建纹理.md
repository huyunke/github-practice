## stb_image.h
### 添加stb_image.h
在项目中加入`stb_image.h`然后在文件头部添加以下代码
```cpp
#define STB_IMAGE_IMPLEMENTATION 
#include "stb_image.h"
```
## 加载与创建纹理
### 声明与绑定
```cpp
unsigned int texture; 
glGenTextures(1, &texture);
glBindTexture(GL_TEXTURE_2D, texture);
```
`glGenTextures`第一个参数：生成纹理的数量

### 设置参数(环绕方式，过滤方式)
在填充数据前，必须告诉 OpenGL 怎么采样：

- **`WRAP_S/T`**: 设置 X/Y 轴超出 $0-1$ 范围时是重复还是拉伸，[纹理环绕方式](纹理环绕方式.md)
    
- **`MIN/MAG_FILTER`**: 设置物体缩小或放大时，像素是取邻近值（锯齿感）还是线性插值（平滑感）[纹理过滤](纹理过滤.md)

### 加载与上传
`glTexImage2D` 执行完后，数据就已经在 GPU 里了
```cpp
int width, height, nrChannels; 
unsigned char *data = stbi_load("container.jpg", &width, &height, &nrChannels, 0);
if (data) {
	glTexImage2D(GL_TEXTURE_2D, 0, GL_RGB, width, height, 0, GL_RGB, GL_UNSIGNED_BYTE, data); 
	glGenerateMipmap(GL_TEXTURE_2D);
}
```
`stbi_load（）`
- 第一个参数：图像文件的位置
- 需要三个`int`作为它的第二、第三和第四个参数，`stb_image.h`将会用图像的**宽度**、**高度**和**颜色通道的个数**填充这三个变量

`glTexImage2D()`
- **`GL_TEXTURE_2D`**: 告诉 OpenGL 这是一个 2D 纹理
    
- **`0`**: 指定 Mipmap 级别。`0` 代表最原始、最高清的那一层
    
- **`GL_RGB`**: **内部格式**。告诉 OpenGL 你希望在显存里以什么格式存储（有些图片带透明度就要用 `GL_RGBA`）
    
- **`width`, `height`**: 纹理的宽度和高度（由 `stbi_load` 获得）
    
- **`0`**: 历史遗留问题，必须填 `0`
    
- **`GL_RGB`**: **源数据格式**。加载进来的图片原始像素格式（stb_image 读进来通常是 RGB）
    
- **`GL_UNSIGNED_BYTE`**: 源数据类型。每个通道是 8 位，即 `unsigned char`
    
- **`data`**: 真正的图像像素数据指针

### 释放内存
```cpp
stbi_image_free(data);
```
**图片数据传给 GPU 后，CPU 里的那块内存（`data`）就没用了，必须释放掉，否则会造成内存泄漏**

## 应用纹理
### 为顶点增加纹理坐标
```cpp
// 纹理属性配置 
glVertexAttribPointer(2, 2, GL_FLOAT, GL_FALSE, 8 * sizeof(float), (void*)(6 * sizeof(float))); 
glEnableVertexAttribArray(2);
```
### 顶点着色器
```
#version 330 core
layout (location = 0) in vec3 aPos;
layout (location = 1) in vec3 aColor;
layout (location = 2) in vec2 aTexCoord; // 接收 location 2 的数据

out vec3 ourColor;
out vec2 TexCoord; // 将坐标发往片段着色器

void main() {
    gl_Position = vec4(aPos, 1.0);
    ourColor = aColor;
    TexCoord = aTexCoord; // 接力传递
}
```

### 片段着色器
```
#version 330 core
out vec4 FragColor;

in vec3 ourColor;
in vec2 TexCoord; // 接收来自顶点着色器的插值坐标

uniform sampler2D ourTexture; // 纹理采样器

void main() {
    // texture() 是内建函数：根据坐标在图片中取色
    FragColor = texture(ourTexture, TexCoord); 
    
    // 如果你想让纹理带上顶点的颜色，可以相乘：
    // FragColor = texture(ourTexture, TexCoord) * vec4(ourColor, 1.0);
}
```

### 绑定纹理(在渲染循环)
在调用`glDrawElements`之前绑定纹理
```cpp
glBindTexture(GL_TEXTURE_2D, texture);
```
