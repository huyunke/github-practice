## 模块的导入与导出
### export,import
- 通过`export`导出一个文件的类、变量、函数等
- 通过`import`导入另一个文件的变量、函数、类等，可以在导入时通过`as`关键词进行重命名

### 重新导出
`export from`用于从一个模块中导出所有的导出项，或从一个模块中导出多个特定的导出项

### 动态import
动态`import`支持条件延迟加载，支持部分反射功能，可以提升页面的加载速度

## instanceof运算符

`instanceof`运算符用于在运行时检查一个对象是否是指定类或其子类的实例
```ArkTS
obj instanceof className
```
## break语句
如果break语句后带有标识符，则将控制流转移到该标识符所包含的语句块之外
```ArkTS
let x = 1;
label: while (true) {
  switch (x) {
    case 1:
      // statements
      break label; // 中断while语句
  }
}
```
